# Smart Expense Tracker

A full-stack web application to help users track income, expenses, and generate category-wise financial reports.

## Tech Stack
- **Backend:** Spring Boot, Hibernate
- **Database:** MySQL
- **Frontend:** React.js (planned)
- **Deployment:** Render (backend), Netlify (frontend planned)

## Features (Backend Completed)
- User registration and authentication
- Add, update, and delete income and expense records
- Category-wise expense tracking
- REST API endpoints for core functionalities
- Email notifications for account activation (if implemented)

## Planned Features
- Frontend UI using React.js
- Daily reminders and notifications
- Export reports to Excel
- Improved dashboard with charts and analytics

## Project Status
- **Backend:** Completed
- **Frontend:** In progress
- Repository structured for full-stack integration

1. Navigate to `backend/` folder
2. Make sure MySQL is running and create a database named `expense_tracker`
